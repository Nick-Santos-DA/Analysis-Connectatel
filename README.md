ConnectaTel Customer Segmentation & Usage Analysis

Project Overview

This project analyzes customer behavior and service usage data for ConnectaTel, a telecommunications company operating in Latin America. The primary objective was to understand how customers interact with mobile services, identify meaningful customer segments, evaluate data quality issues, and generate business recommendations to support customer retention and service optimization.

The analysis combines data cleaning, exploratory data analysis (EDA), customer profiling, segmentation techniques, statistical summaries, outlier detection, and data visualization to transform raw operational data into actionable business insights.

⸻

Business Problem

ConnectaTel needed to better understand how customers use their services and whether different customer groups exhibit distinct usage patterns.

Key business questions included:

* How is the customer base distributed by age and service usage?
* Are there identifiable customer segments with different behaviors?
* Which customers represent the highest business value?
* Are there unusual usage patterns that could indicate opportunities or risks?
* How can current service plans be optimized to better serve different customer profiles?

⸻

Dataset Description

The project used three datasets:

Plans

Contains information about available service plans, including:

* Monthly fees
* Included messages
* Included call minutes
* Additional usage charges

Users

Contains customer demographic and subscription information:

* Customer ID
* Age
* City
* Registration date
* Plan type
* Churn information

Usage

Contains historical service usage records:

* Calls
* Messages
* Usage dates
* Call duration

⸻

Data Cleaning & Preparation

Several data quality issues were identified and addressed:

* Missing values in customer location data.
* Future registration dates that required correction.
* Sentinel values in age records.
* Missing values in call duration and message length variables.
* Validation of missing data mechanisms (MAR analysis).
* Data type corrections and date conversions.

A thorough validation process ensured that the final dataset was reliable for analysis and segmentation.

⸻

Exploratory Data Analysis

The analysis included:

* Descriptive statistics.
* Distribution analysis.
* Histograms and boxplots.
* Outlier detection using the IQR method.
* Missing value assessment.
* Customer usage aggregation.

Usage metrics were aggregated at the customer level to build comprehensive customer profiles.

⸻

Customer Segmentation

Customers were segmented according to:

Usage Level

* Low Usage
* Medium Usage
* High Usage

Segmentation was based on the number of calls and messages generated by each customer.

Age Group

* Young (<30 years)
* Adult (30–59 years)
* Senior Adult (60+ years)

These segments allowed for a deeper understanding of customer behavior and service consumption patterns.

⸻

Key Findings

* Most customers belong to the Adult age segment.
* Medium Usage customers represent the majority of the customer base.
* High Usage customers form a smaller but strategically valuable segment.
* Several usage outliers were identified, particularly in total call minutes, representing highly engaged users rather than data quality issues.
* Customer behavior is generally concentrated around moderate service consumption levels.

⸻

Business Recommendations

Based on the analysis, the following recommendations were proposed:

* Develop premium plans targeted at high-usage customers.
* Create lower-cost plans for low-usage customers to improve retention.
* Implement targeted marketing campaigns focused on the largest customer segments.
* Monitor high-consumption users for upselling opportunities.
* Strengthen data quality controls to prevent future inconsistencies.
